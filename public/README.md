# Public Assets

This directory contains static assets served by the application.

## Favicon

- **favicon.svg** - SVG favicon with PolyForge branding (32x32, scalable)
  - Purple gradient background (#667eea to #764ba2)
  - White "PF" text (PolyForge initials)
  - Modern, clean design

### Alternative Favicon Generation

If you need a traditional .ico file, open `generate-favicon.html` in your browser:

1. Open `generate-favicon.html` in a web browser
2. Click "Download favicon.ico"
3. Move the downloaded file to this directory
4. Update `index.html` to reference `favicon.ico` instead of `favicon.svg`
