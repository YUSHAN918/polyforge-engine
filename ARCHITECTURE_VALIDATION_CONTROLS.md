# 🎬 架构验证观测窗口 - 导演级控制面板

## 新增功能

### 🎨 后期特效控制

#### 辉光强度 (Bloom Strength)
- **范围**: 0.0 - 3.0
- **步长**: 0.1
- **默认值**: 1.5
- **效果**: 实时调整辉光效果强度
- **接口**: `window.renderDemoControls.setBloomStrength(value)`

#### 辉光阈值 (Bloom Threshold)
- **范围**: 0.0 - 1.0
- **步长**: 0.05
- **默认值**: 0.5
- **效果**: 控制哪些亮度的物体会发光
- **接口**: `window.renderDemoControls.setBloomThreshold(value)`

---

### 🌅 环境导演控制

#### 时间 (Time of Day)
- **范围**: 0 - 24 小时
- **步长**: 0.1
- **默认值**: 17:00 (日落前1小时)
- **效果**: 拖拽滑块即可看到日出日落
- **接口**: `manager.setTimeOfDay(hour)`

#### 光照强度 (Sun Intensity)
- **范围**: 0.0 - 5.0
- **步长**: 0.1
- **默认值**: 1.0
- **效果**: 实时调整太阳光照强度，可以"熄灭太阳"
- **接口**: `manager.setLightIntensity(intensity)`

---

## 🔧 修复问题

### 地形材质亮度溢出
- **问题**: 创建山峰后，地形过亮，像"发光的太阳"
- **修复**: 将地形材质的 `roughness` 从 0.9 调整为 0.8
- **位置**: `ArchitectureValidationManager.ts` 初始化场景

---

## 🎮 使用方法

1. 打开架构验证观测窗口
2. 在右侧面板找到"后期特效"和"环境导演"滑块组
3. 拖动滑块，效果实时生效，无需刷新页面

### 推荐调试组合

**日落场景**:
- 时间: 18:00
- 光照强度: 0.8
- 辉光强度: 2.0
- 辉光阈值: 0.3

**午夜场景**:
- 时间: 0:00
- 光照强度: 0.1
- 辉光强度: 1.0
- 辉光阈值: 0.8

**正午场景**:
- 时间: 12:00
- 光照强度: 3.0
- 辉光强度: 0.5
- 辉光阈值: 0.7

## 🎨 UI 设计

- **深色主题**: 所有滑块在深色背景下清晰可见
- **实时反馈**: 滑块旁显示当前数值
- **颜色编码**:
  - 紫色: 后期特效
  - 黄色: 环境控制
  - 绿色: 地形控制
- **渐变滑块**: 使用渐变色滑块手柄，带发光效果
